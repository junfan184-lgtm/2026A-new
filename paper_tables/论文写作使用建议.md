# A题论文写作使用建议

## 数据与预处理部分

- 使用 `表1_变量说明.csv` 说明变量类别、单位和建模含义。
- 使用 `表2_数据概况与异常统计.csv` 说明样本规模、缺失值和CO异常候选点。
- 可引用 `02_CO异常候选点.csv` 说明序列1044-1059附近存在非稳态/检测异常候选段。

## 问题一：风箱温度预测

- 主表使用 `表3_问题一温度预测模型对比.csv`。
- 结论写法：温度状态-相邻负压滞后模型显著优于仅负压滞后模型，平均R2达到0.807。
- 细节表使用 `表4_各风箱最优温度预测结果.csv`。

## 问题二：CO浓度预测

- 主表使用 `表5_CO预测模型对比.csv`。
- 实时监测模型：采用含CO历史滞后项的随机森林，R2为0.838。
- 调控优化模型：采用不含CO历史项的GradientBoosting，保持控制变量解释性。
- 滞后依据可引用 `表6_滞后相关Top15.csv`。

## 问题三：负压优化

- 优化效果表使用 `表7_典型工况优化效果.csv`。
- 约束范围表使用 `表8_风箱负压调控范围.csv`。
- 推荐方案表使用 `表9_典型工况推荐负压汇总.csv`。
- 图可使用 `../figures/06_典型工况推荐负压曲线.png`。

## 稳健性分析

- 使用 `表_异常处理稳健性.csv` 说明异常剔除后模型表现更稳定。
- 使用 `表_工艺变量滞后敏感性.csv` 说明不同工艺滞后阶数下结果接近。
- 使用 `表_CO历史滞后项敏感性.csv` 说明CO历史项对在线监测有显著贡献。
- 图可使用 `../figures/07_稳健性分析RMSE对比.png`。
